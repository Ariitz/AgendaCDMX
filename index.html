🔧 Configurar Firebase (base de datos gratis)

  1. Ve a console.firebase.google.com
  2. Crea un proyecto nuevo (nombre libre, p.ej. "mi-agenda-cdmx")
  3. En el menú lateral: Build → Firestore Database → Create database
  4. Selecciona Start in test mode (30 días gratis, puedes ampliar)
  5. Ve a Project Settings ⚙️ → General → Your apps → Add app → Web </>
  6. Copia el objeto firebaseConfig que te da Google
  7. Pégalo en el archivo HTML reemplazando el bloque FIREBASE_CONFIG al inicio del <script type="module">
  8. Guarda el archivo y vuelve a subirlo a Vercel

✅ Plan gratuito de Firebase incluye:

1 GB de almacenamiento · 50,000 lecturas/día · 20,000 escrituras/día · más que suficiente para uso personal

⚠️ Sin Firebase configurado

Los planes y favoritos se guardan solo en este navegador/dispositivo con localStorage. Funcionan perfectamente, pero no se sincronizan entre dispositivos.
